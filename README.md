# Installation

Add the dependency:

```ruby
npm i react-native-emoji-slider
```

## Peer Dependencies

###### IMPORTANT! You need install them

```js
"react": ">= 16.x.x",
"react-native": ">= 0.55.x",
"@miblanchard/react-native-slider": "^1.1.0"
```

# Usage

## Import

```js
import EmojiSlider from "react-native-emoji-slider";

<EmojiSlider />;
```

## EmojiSlider Usage

```js
<EmojiSlider
  onChange={(res) => {
    console.log("res:", res);
  }}
/>
```
